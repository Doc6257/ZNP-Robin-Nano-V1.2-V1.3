# ZNP-Robin-Nano-V1.2
ZNP Robin Nano v1.2 is a powerful 32-bit 3D printer control board with STM32F103VET6. Support Marlin2.0. Support ZNP Robin TFT35 V1.0 touch Screens. The mainboard integrates 5 AXIS interface, hot bed, heating head, 3 NTC100K, 1 MAX31855.The motherboard integrates After shutdown. Support Robin WIFI for cloud printing and supports firmware update by SD card…



# About ZNP-Robin-Nano V1.2 and V1.3
The differences between ZNP Robin Nano V1.2 and ZNP Robin Nano V1.3 are as follow:


```
————————————————————————————————————————————————————————————————————————————————————————————————————————
                        ZNP Robin Nano V1.2                                   ZNP Robin Nano V1.3            
————————————————————————————————————————————————————————————————————————————————————————————————————————
Firmware support        ZNP Robin Nano V1.2 Firmware/Above Marlin2.0.6        ZNP Robin Nano V1.3/Above Marlin2.0.6    
————————————————————————————————————————————————————————————————————————————————————————————————————————
LCD Type and Interface  ZNP Robin TFT35(FSMC)                                 ZNP Robin TFT35(FSMC)    
————————————————————————————————————————————————————————————————————————————————————————————————————————
Rotary Encoder          Not support                                           Not support  
————————————————————————————————————————————————————————————————————————————————————————————————————————
Motor Driver mode       STEP/DIR                                              STEP/DIR   
————————————————————————————————————————————————————————————————————————————————————————————————————————
On board EEPROM         2KB                                                   4KB                          
————————————————————————————————————————————————————————————————————————————————————————————————————————
```
